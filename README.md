# Портфолио UI/UX (HTML + CSS)

Минимальный статический каркас сайта: главная страница + детальные страницы проектов.

## Что открыть
- Главная: `index.html`
- Детальные проекты:
  - `projects/one/index.html`
  - `projects/two/index.html`

## Где менять контент
- Имя, роль, ссылки, текст: `index.html`
- Заголовок/описание/блоки: файлы в `projects/`

## Картинки
Сейчас используются плейсхолдеры:
- `assets/me.png`
- `assets/project-one-thumb.jpg`, `assets/project-two-thumb.jpg`, и т.д.

Создай папку `assets/` и положи туда свои изображения с теми же именами (или поменяй пути в HTML).

## Хостинг
Такой сайт легко публикуется “как есть”:
- GitHub Pages (если захочешь)
- Netlify / Vercel
- любой обычный static hosting

Пока без сборщиков и без сложного JS — чтобы было максимально просто начать.
